# Crowdfunding dapp example

## Deploy a smart contracts
**Be in the root of this repo**
1) Install Truffle:
```bash
npm install -g truffle
```
2) Install dependencies:
```bash
npm install
```
3) Setup deploy environment:
```bash
bash ./scripts/setup.sh
```
4) Truffle compile:
```bash
truffle compile
```
5) Deploy:
```bash
bash ./scripts/deploy.sh
```
